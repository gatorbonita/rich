# Portfolio Optimizer Frontend

React + Vite frontend for the portfolio optimization web app.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will be available at http://localhost:5173

## Build

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Features

- Interactive portfolio configuration
- Real-time optimization
- Beautiful visualizations (sector allocation, correlation heatmap)
- Responsive design
- Alternative portfolio suggestions
- Stock details with risk metrics

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Recharts
- Axios
